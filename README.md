# Plateforme-de-Gestion-Hospitali-re-HSP---Project-examen-BTS-JavaFx
Application de gestion pour service d'urgence hospitalier développée en JavaFX. Gère les patients (fiches, dossiers, hospitalisations), les stocks de produits médicaux et les utilisateurs (secrétaires, médecins, gestionnaires). Base de données MySQL avec traçabilité des actions conforme RGPD.
